Detailed Description

The function declaration for the I2CM callback is:

void callback(i2cm_req_t * req, int error_code);
req Pointer to an i2cm_req object representing the I2CM active transaction.
error_code An error code if the active transaction had a failure or E_NO_ERROR if successful.

§ I2CM_Handler()

void I2CM_Handler ( mxc_i2cm_regs_t *  i2cm)

This function is an IRQ handler and will be called by the core if I2CM interrupts are enabled. Alternately, if the application is using asynchronous methods, this function can be periodically called by the application if the I2CM interrupts are disabled.

Parameters
i2cmBase address of the I2CM module.
