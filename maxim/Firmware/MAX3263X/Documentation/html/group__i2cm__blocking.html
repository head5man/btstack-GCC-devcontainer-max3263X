§ I2CM_Read()

int I2CM_Read ( mxc_i2cm_regs_t *  i2cm,
uint8_t  addr,
const uint8_t *  cmd_data,
uint32_t  cmd_len,
uint8_t *  data,
uint32_t  len 
)

Will block until transaction is complete.

Note
Command is an optional feature where the master will write the cmd_data before reading from the slave. If command is undesired, set the *cmd_data parameter to NULL and pass 0 for the cmd_len parameter.
If there is a command, the master will send a repeated start sequence before attempting to read from the slave.
This function blocks until the transaction has completed.
Parameters
i2cmPointer to the I2CM registers, see mxc_i2cm_regs_t.
addrI2C address of the slave.
cmd_dataData to write before reading.
cmd_lenNumber of bytes to write before reading.
dataWhere to store the data read.
lenNumber of bytes to read.
Returns
Number of bytes read if successful, error code if unsuccessful.

Note
Command is an optional feature where the master will write the cmd_data before writing the data to the slave. If command is not needed, set the cmd_data to NULL and set cmd_len to 0. If there is a command, the master will send a repeated start sequence before attempting to read from the slave.
This function blocks until the transaction has completed.
Parameters
i2cmPointer to the I2CM registers, see mxc_i2cm_regs_t.
addrI2C address of the slave.
cmd_dataData to write before writing data.
cmd_lenNumber of bytes to write before writing data.
dataData to be written.
lenNumber of bytes to Write.
Returns
Number of bytes writen if successful or an Error Code if unsuccessful.
