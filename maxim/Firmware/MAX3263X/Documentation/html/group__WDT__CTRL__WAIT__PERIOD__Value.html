Detailed Description

The WAIT_PERIOD field sets the duration of the watchdog pre-window period. If the watchdog is reset before the wait period has finished, an out-of-window interrupt will occur. This sets the minimum amount of time between watchdog enable/clear to resetting the WDT count and assists in detecting overclocking or an invalid clock. The values defined are in the number of watchdog clock cycles.
