§ LP_EnterLP0()

void LP_EnterLP0 ( void  )
Note
Important notes for LP0 mode entry and exit
  • SRAM contents are lost
  • Waking up from LP0 is similar to a system reset, code execution begins from address 0
  • This function does not return.
Note
Interrupts are globally disabled upon entering this function.
