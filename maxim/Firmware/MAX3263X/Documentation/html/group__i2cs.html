§ i2cs_addr_t

enum i2cs_addr_t
Enumerator

Sets the slave address mode to 8-bits (7-bits address plus read/write bit).

I2CS_ADDR_10 

Sets the slave address mode to 10-bits.

§ I2CS_Handler()

void I2CS_Handler ( mxc_i2cs_regs_t *  i2cs)

This function should be called by the application from the interrupt handler if I2CS interrupts are enabled. Alternately, this function can be periodically called by the application if I2CS interrupts are disabled.

Parameters
i2csPointer to I2CS regs.

§ I2CS_RegisterCallback()

void I2CS_RegisterCallback ( mxc_i2cs_regs_t *  i2cs,
uint8_t  addr,
i2cs_callback_fn  callback 
)

Registering a callback causes the slave to interrupt when the master has updated a specified byte.

Parameters
i2csPointer to the I2CS register structure, see mxc_i2cs_regs_t.
addrIndex to trigger a call to the i2cs_callback_fn.
callbackcallback function of type i2cs_callback_fn to be called when the addr being written by the master matches addr.
