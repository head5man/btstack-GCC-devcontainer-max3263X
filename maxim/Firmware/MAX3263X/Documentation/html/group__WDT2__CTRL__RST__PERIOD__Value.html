Detailed Description

The RST_PERIOD field sets the duration of the watchdog reset period, which is the time period from the WDT being enabled/cleared until the WDT2 flag, MXC_F_WDT2_CTRL_RST_PERIOD is set. The values defined are in the number of watchdog clock cycles.
