§ uart_async_callback

typedef void(* uart_async_callback) (uart_req_t *, int)

Type alias uart_async_callback for a callback function with signature of:

void callback_fn(uart_req_t *request , int error_code);

uart_req_t * Pointer to the transaction request. error_code Return code for the UART request. See Error Codes for possible codes.

§ UART_Handler()

void UART_Handler ( mxc_uart_regs_t *  uart)

This function should be called by the application from the interrupt handler if UART interrupts are enabled. Alternately, this function can be periodically called by the application if UART interrupts are disabled. Only necessary to call this when using asynchronous functions.

Parameters
uartPointer to the UART registers.
