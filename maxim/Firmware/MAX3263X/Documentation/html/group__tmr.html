§ tmr_prescale_t

enum tmr_prescale_t

The prescaler divides the peripheral input clock to the timer by a selectable divisor.

Enumerator
TMR_PRESCALE_DIV_2_0 

Divide the peripheral input clock by \( TMR_{Prescaler} = 2^{0} = 1 \).

TMR_PRESCALE_DIV_2_1 

Divide the peripheral input clock by \( TMR_{Prescaler} = 2^{1} = 2 \).

TMR_PRESCALE_DIV_2_2 

Divide the peripheral input clock by \( TMR_{Prescaler} = 2^{2} = 4 \).

TMR_PRESCALE_DIV_2_3 

Divide the peripheral input clock by \( TMR_{Prescaler} = 2^{3} = 8 \).

TMR_PRESCALE_DIV_2_4 

Divide the peripheral input clock by \( TMR_{Prescaler} = 2^{4} = 16 \).

TMR_PRESCALE_DIV_2_5 

Divide the peripheral input clock by \( TMR_{Prescaler} = 2^{5} = 32 \).

TMR_PRESCALE_DIV_2_6 

Divide the peripheral input clock by \( TMR_{Prescaler} = 2^{6} = 64 \).

TMR_PRESCALE_DIV_2_7 

Divide the peripheral input clock by \( TMR_{Prescaler} = 2^{7} = 128 \).

TMR_PRESCALE_DIV_2_8 

Divide the peripheral input clock by \( TMR_{Prescaler} = 2^{8} = 256 \).

TMR_PRESCALE_DIV_2_9

§ TMR_Init()

int TMR_Init ( mxc_tmr_regs_t *  tmr,
tmr_prescale_t  prescale,
const sys_cfg_tmr_t *  sysCfg 
)

This function initializes the timer to a known state and saves the prescaler. The timer will be left disabled. TMR_Init should be called before TMR_Config.

Parameters
tmrPointer to timer registers for the timer instance to modify.
prescaleclock divider for the timer clock
sysCfgPointer to the timer system GPIO configuration. If not using GPIO for the timer instance, set this parameter to NULL.
Return values
E_NO_ERRORTimer initialized successfully.
ErrorCode Timer was not initialized, see Error Codes.
