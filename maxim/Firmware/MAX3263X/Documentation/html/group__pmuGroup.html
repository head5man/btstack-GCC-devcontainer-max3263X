Typedef Documentation

§ pmu_callback

typedef void(* pmu_callback) (int pmu_status)

The callback function signature is:

void callback(int status);

pmu_status - The callback function argument is a status bit indicating the status of the PMU program. The callback function will be called for every opcode that has the interrupt bit set. If NULL, the channel interrupt will not be enabled.
