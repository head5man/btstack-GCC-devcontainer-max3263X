§ gpio_pad_t

enum gpio_pad_t
Enumerator
GPIO_PAD_INPUT_PULLUP 

Set pad to high impedance, weak pull-up.

GPIO_PAD_OPEN_DRAIN 

Set pad to open-drain with high impedance with input buffer.

GPIO_PAD_OPEN_DRAIN_PULLUP 

Set pad to open-drain with weak pull-up.

GPIO_PAD_INPUT 

Set pad to high impednace, input buffer enabled.

GPIO_PAD_NORMAL 

Set pad to normal drive mode for high an low output.

GPIO_PAD_SLOW 

Set pad to slow drive mode, which is normal mode with negative feedback to slow edge transitions.

GPIO_PAD_FAST 

Set pad to fash drive mode, which is normal mode with a transistor drive to drive fast high and low.

GPIO_PAD_INPUT_PULLDOWN 

Set pad to weak pulldown mode.

GPIO_PAD_OPEN_SOURCE 

Set pad to open source mode, transistor drive to high.

GPIO_PAD_OPEN_SOURCE_PULLDOWN 

Set pad to open source with weak pulldown mode, transistor drive to high, weak pulldown to GND for low.
